# 小程序试用清单

麻烦您照着下面一条条操作，看看**实际看到的**和**预期**是不是一样。


---

## 一、第一次使用

| 操作 | 预期看到 | 实际情况 | 是否正常 |
|---|---|---|---|
| 打开小程序 | 进入首页，顶上是问候语（早上好/下午好），下面有「学员/充值/课表/统计」四个按钮，右上角有一个齿轮和一个小铃铛 | | |
| 点「学员」→ 添加一个学员（比如「张三」） | 学员列表里出现张三，显示剩余 0 课时 | | |
| 点「充值」→ 给张三充 10 次 | 提示「剩余课时 0 → 10 次」，张三的剩余课时变成 10 | | |
| 点右上角齿轮 → 进「设置」看「课程类型」 | 看到已有「补课」「剑桥课程」两种，可以在这里加自己的课型 | | |
| 在设置里加一个课型（名称「自然拼读」，时长 60 分钟）→ 点添加 | 列表里出现「自然拼读 60 分钟」 | | |

## 二、日常排课

| 操作 | 预期看到 | 实际情况 | 是否正常 |
|---|---|---|---|
| 进「课表」→ 点右下角加号 → 新建一节课：选课型、选张三、选今天某个时间 → 创建 | 提示「课程已创建」，课表里今天那天出现这节课，写着课型和张三 | | |
| 再建一节「循环课」：切到循环，勾周一、周三，选开始和结束日期 | 提交前显示「将生成 N 节课」 | | |
| 提交这节循环课 | 提示已生成 N 节课，课表里多个日期都能看到这些课 | | |
| 在课表上点左右箭头翻到下一周 / 上一周 | 顶部日期范围跟着变，显示对应那一周的课 | | |
| 点开一节还没上的课 → 改一下上课时间 → 保存 | 课表里这节课挪到了新时间 | | |
| 点开一节还没上的课 → 取消 | 这节课变成「已取消」状态 | | |

## 三、学生上课后

| 操作 | 预期看到 | 实际情况 | 是否正常 |
|---|---|---|---|
| 点开张三今天已经上的课 → 标记「完成」 | 这节课变成「已完成」，张三的剩余课时少 1（比如 10 → 9） | | |
| 点右上角铃铛进「消息」 | 里面出现一条「扣课时：张三 扣 1 课时，剩余 9」 | | |
| 找一节刚标完成的课 → 点「撤销」 | 课回到「待上课」，张三的课时退回来（9 → 10） | | |
| 一节有多个学生的小班课 → 标记完成时，把没来的那个学生取消勾选 | 只给来了的学生扣课时，没来的不扣 | | |
| 一节课整节没上 → 标记「缺勤」 | 课变成「缺勤」，谁的课时都不扣 | | |

## 四、课时和提醒

| 操作 | 预期看到 | 实际情况 | 是否正常 |
|---|---|---|---|
| 把某个学生的课时用到只剩 2 次或更少 | 首页出现「课时不足」的提醒区块，列着这个学生和剩余次数 | | |
| 进「消息」看这条课时不足 | 这条消息整张卡片是**橘红色**的（和普通消息不一样，一眼能看出要处理） | | |
| 给这个学生再充值 | 提醒消失（或次数更新），消息里出现一条充值记录 | | |
| 点右上角铃铛顶部的「开启自动提醒」→ 按提示允许 | 之后上课前、课时不够时，手机能收到提醒（**注意：这个只有真机能收到，用电脑模拟看不到**） | | |

## 五、查看经营情况

| 操作 | 预期看到 | 实际情况 | 是否正常 |
|---|---|---|---|
| 点「统计」 | 看到一块像日历格子的图（上课多的日子颜色深），还有本周的课时数据 | | |
| 点开某个学员 | 看到他的剩余课时、充值/扣课时记录、以及他上过和要上的课 | | |

## 六、故意用错，看看会不会拦住

| 操作 | 预期看到 | 实际情况 | 是否正常 |
|---|---|---|---|
| 添加学员时**不填名字**直接提交 | 提示要填名字，不会创建一个空白学员 | | |
| 给**同一个学生**在**同一个时间段**排两节课 | 弹出「该学生在这个时段已经有课，无法重复排」，而且**没有**「仍要创建」的选项 | | |
| 给**两个不同学生**排在**同一个时间段** | 弹出时段冲突提示，但**允许**你选择「仍要创建」（可能是你有意安排的） | | |
| 给一个**剩余 0 课时**的学生排课，并标记完成 | 能排、能完成，但完成后他的课时会变成 -1，首页也会把他列进课时不足提醒 | | |
| 在设置里**删除一个已经用过的课型**（比如补课） | 提示这个课型已经被课程用过，无法删除，建议改成「停用」 | | |
| 充值时填 0 或负数 | 提示充值次数要填正整数，不会充成功 | | |

---

## 请你特别留意（这几处我们心里没底，想听你真实感受）


1. **顺不顺手**：新建课、充值、标记完成这几件最常做的事，操作步骤是不是符合你的习惯？有没有点了一下不知道下一步去哪？
2. **找不找得到**：有没有「我想做某件事，但不知道在哪点」的时候？（比如想改一节课、想看某个学生以前上过的课）
3. **看不看得清**：手写风格的字、颜色、卡片，读起来清楚吗？有没有觉得哪里太小、太淡、看不清？
4. **提醒有没有用**（真机）：手机收到的上课提醒、课时不足提醒，内容说得清楚吗？会不会太多太吵，或者该来的没来？
5. **消息中心**：里面的消息对你有用吗？会不会太多？有没有你希望有、但现在没有的提醒？
